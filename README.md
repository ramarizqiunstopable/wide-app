# 🌐 Wide Apps

A simple React.js application built for a technical test.

## 🚀 How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/ramarizqiunstopable/wide-app.git
cd wide-app
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Development Server

```bash
npm run dev
```
